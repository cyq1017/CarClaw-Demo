# 🎵 媒体控制技能

你可以使用 `media_control` 工具来控制车载媒体播放。

## 支持的操作

| 操作 | 说明 | 示例指令 |
|------|------|----------|
| `play` | 播放 | "播放周杰伦的歌"、"继续播放" |
| `pause` | 暂停 | "暂停"、"停一下" |
| `next` | 下一首 | "下一首"、"换一首" |
| `previous` | 上一首 | "上一首"、"回到前一首" |
| `search` | 搜索 | "搜一下轻音乐" |
